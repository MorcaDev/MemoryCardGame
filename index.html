This is a memory card game with 8 levels.

Here you need to remember an specific kind of card , and obviously you'll choose the type of them :)

When the game starts,You will see the cards for a few seconds

In this Section you will able to set your kind of cards
Set ot before you start the game

There are only 2 buttons : "START" button starts the game only one time , and "NEW GAME" starts from lvl 1 again if you lose the game
